# Koffy v0.2.1 Release Notes

Koffy v0.2.1 is a small patch release focused on recharge payment experience.

## Fixes

- Improved Alipay payment from the recharge page in regular mobile and desktop browsers: Alipay now opens in a separate page, so the Koffy recharge page remains available for checking order status.
- Added a WeChat in-app browser notice when users choose Alipay, recommending WeChat Pay for the best in-WeChat experience and explaining the manual copy-link fallback for Alipay.
- Kept Alipay fallback inside the current WeChat embedded browser page to avoid the blank-page behavior caused by opening an intermediate empty page in WeChat.

## Verification

- `npm run build`
